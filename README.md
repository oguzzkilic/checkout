# Payment Checkout Form

## Requirements
- Npm or Yarn
- Node.js (8.9.1 >)

## Install from NPM
```sh
npm install
```

## How Does it Work?
```sh
npm run start
npm run test
npm run lint
```

## Techs
- webpack (module bundler)
- babel (es6-7 to es5)
- react
- redux
- redux-thunk (side-effect middleware)
- redux-form
- styled-components (css in JavaScript)
- jest (unit test)
- airbnb enzyme (unit test)
- eslint (JavaScript lint)
- prettier (code formatter)
